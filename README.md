## ANGULAR

### Pré - requisitos
  Antes de instalar o Angular, você precisa ter o Node.js e o npm (gerenciador de pacotes) instalados.

### Instalação do Angular

  Para iniciar a instalação do Angular, é preciso fazer a instalação da ferramenta que permite criar, gerenciar e buildar aplicações Angular via terminal que é: npm install -g @angular/cli.

  Verifique se o ng foi instalado (extensão do Angular) utilizando o código: ng version

### Execução no Projeto

  A execução do Angular no projeto é feita de maneira muito simples. Dentro do arquivo do seu projeto abra o terminal e comece a criação: 

Crie o seu Projeto
*ng new meu-primeiro-app-angular*

### Por que utilizar o Angular?

O Angular (mantido pelo Google) é um dos frameworks mais robustos do ecossistema JavaScript. Ele não é apenas uma biblioteca de interface (como o React), mas sim uma plataforma completa (framework "opinionated"), o que significa que ele já traz quase todas as ferramentas necessárias nativamente.
